cosmetic($module, $values)
cosmetic($module, $property, $values)

Creates a cosmetic module for setting a single property.

If the $property is not provided, it is supposed to be equal to the module name.

The $values could be either list or map. In the first case, the suffix of the module name would be the same as value, otherwise an itemâ€™s key stands for it.

@include cosmetic('float', (left, right));
@include cosmetic('margin', ('m': 32px, 'l': 128px));
@include cosmetic('valign', 'vertical-align', (bottom, top));

CSS output

.float-left {
  float: left !important;
}

.float-right {
  float: right !important;
}

.margin-m {
  margin: 32px !important;
}

.margin-l {
  margin: 128px !important;
}

.valign-bottom {
  vertical-align: bottom !important;
}

.valign-top {
  vertical-align: top !important;
}

font-face($family, $filename, $weight: normal, $style: normal)

Allows easier custom fonts integration.

@include font-face('Open Sans', 'OpenSans-Regular');

CSS output

@font-face {
  font-family: 'Open Sans';
  src: url("/assets/fonts/OpenSans-Regular.eot");
  src:
    url("/assets/fonts/OpenSans-Regular.eot?#iefix") format('embedded-opentype'),
    url("/assets/fonts/OpenSans-Regular.woff") format('woff'),
    url("/assets/fonts/OpenSans-Regular.ttf") format('truetype'),
    url("/assets/fonts/OpenSans-Regular.svg#Open%20Sans") format('svg');
  font-weight: normal;
  font-style: normal;
}

font-height($font-size, $line-height)

Shorthand notation for setting font metrics of an element.

@include font-height(12px, 1.62);

CSS output

font-size: 12px;
line-height: 1.62;

hidpi-image($image-path)

Includes background-image and background-size for the named HIDPI image.

hidpi-inline($image-path)

Embeds background-image and include background-size for the named HIDPI image.

image-size($image-path, $factor: 1)

Sets the width and height of an element to the dimensions of the named image.

The second paramater is a divisor which is applied onto both dimensions.

position($position, $coordinates: none)
position($coordinates)

Shorthand notation for setting position of an element.

The first parameter is optional and sets position of an element (absolute, relative, etc.)

The second parameter is a space delimited list of offsets (top, right etc.) that follows the standard CSS shorthand notation.

Offsets passed as none are not being output.

@include position(absolute, 12px 0 none);

CSS output

position: absolute;
top: 12px;
right: 0;
left: 0;

reject($property, $value, $filters)

Includes property only if it's value is not in the list of filters.

size($size)

Shorthand notation for setting size of an element.

The only parameter is a space delimited width and height. If height is omitted, it is considered to equal width.

Dimensions passed as none are not being output.
