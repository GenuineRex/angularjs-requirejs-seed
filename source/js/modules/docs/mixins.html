font-height($font-size, $line-height)

Shorthand notation for setting font metrics of an element.

@include font-height(12px, 1.62);

CSS output

font-size: 12px;
line-height: 1.62;

hidpi-image($image-path)

Includes background-image and background-size for the named HIDPI image.

hidpi-inline($image-path)

Embeds background-image and include background-size for the named HIDPI image.

image-size($image-path, $factor: 1)

Sets the width and height of an element to the dimensions of the named image.

The second paramater is a divisor which is applied onto both dimensions.

position($position, $coordinates: none)
position($coordinates)

Shorthand notation for setting position of an element.

The first parameter is optional and sets position of an element (absolute, relative, etc.)

The second parameter is a space delimited list of offsets (top, right etc.) that follows the standard CSS shorthand notation.

Offsets passed as none are not being output.

@include position(absolute, 12px 0 none);

CSS output

position: absolute;
top: 12px;
right: 0;
left: 0;

reject($property, $value, $filters)

Includes property only if it's value is not in the list of filters.

size($size)

Shorthand notation for setting size of an element.

The only parameter are space delimited width and height. If height is omitted, it is considered to equal width.

Dimensions passed as none are not being output.
