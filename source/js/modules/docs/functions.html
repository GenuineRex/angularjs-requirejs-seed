Functions

asset-url($filename)

Returns an absolute URL to the named file.

asset-url('flowers.png') //=> url('/assets/images/media/flowers.png')

escape-css($data)

Returns a string containing the escaped version of the provided data for use as a CSS identifier.

escape-css('1a2b3c') //=> '#\31 a2b3c'

escape-url($data)

Returns a string containing the escaped version of the provided data for use as an URL identifier.

escape-url('100% Love') //=> '100%25+Love'

image-height($filename)

Returns the height of the named image.

image-height('flowers.png') //=> 240px

image-width($filename)

Returns the width of the named image.

image-width('flowers.png') //=> 320px

inline-asset($filename)

Embeds the contents of the named asset directly inside a stylesheet.

inline-asset('flowers.png') //=> url('data:image/png;base64,...')

inline-stroke($color, $spacing: 0)

Returns a base64-encoded picture of a stoke described by the passed properties.

inline-stroke(rgba(#C00, 0.2), 2px) //=> url('data:image/png;base64,...')

unpack($shorthand, $length: 4)

Unpacks the named list according to the standard CSS shorthand notation.

The second parameter is target length of an unpacked list.

unpack(5px 2px) #=> 5px 2px 5px 2px
unpack(100%, 2) //=> 100% 100%
